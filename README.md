# 🤖 FrontendGen AI — AI Frontend Generator

FrontendGen AI is an AI-powered tool that generates frontend UI layouts from prompts. Users can modify generated interfaces using additional prompts, switch between versions, and manage usage through a credit-based system.

This project demonstrates AI integration with modern frontend workflows and version-based UI generation.

---

## 🚀 Live Demo

🔗 Live App: https://buildify-gold.vercel.app/ site builder

---

## ✨ Features

- 🤖 Generate frontend UI from text prompts  
- ✏️ Update generated UI using additional prompts  
- 🔄 Switch between previous UI versions  
- 💳 Credit-based system for feature usage  
- ⚡ Fast UI generation workflow  
- 📦 Version control-like UI rollback system

---

## 🧠 How It Works

1. User enters a prompt describing the desired UI  
2. AI generates a frontend layout  
3. Users update the layout using new prompts  
4. Previous versions are stored for switching back  
5. Credits are deducted based on feature usage
